# 🧪 Виртуальная лаборатория БЖД

Интерактивная виртуальная лаборатория по **Безопасности жизнедеятельности** с 3D-визуализацией и реальными физическими расчётами производственных факторов.

## 📋 Описание

Веб-приложение для изучения основ промышленной безопасности через:
- Интерактивные 3D-сцены с реалистичной физикой
- Пошаговое изучение теории с формулами
- Тестирование знаний
- Полноценные лабораторные работы с расчётами

Приложение охватывает 5 основных направлений: производственное освещение (исследование и расчёт), шум и вибрация, защита от электромагнитных излучений (ЭМИ), расчёт напряжённости поля УВЧ-диапазона.

## 🎓 Лабораторные работы

### 1️⃣ Лаборатория №1 — Исследование освещённости
**Цель:** изучить зависимость освещённости от параметров источника света.

**Формулы:**
- Освещённость: `E = I / r²` (лк), где `r = √(h² + d²)`
- Коэффициент естественной освещённости: `КЕО = (E_в / E_н) × 100%`

**Интерактивность:**
- Регулировка высоты источника (0.5–4.5 м)
- Изменение силы света I (100–5000 кд)
- Перемещение точки измерения по горизонтали (0–3 м)
- Таблица сравнения с нормами (точная работа, офис, коридоры)

### 2️⃣ Лаборатория №2 — Расчёт освещённости
**Цель:** рассчитать количество светильников методом коэффициента использования.

**Формулы:**
- Индекс помещения: `i = L×B / (Hp × (L+B))`
- Количество светильников: `N = (E_н × S × K_з × z) / (n × Ф_л × η)`

**Интерактивность:**
- Задание размеров помещения (L, B, H)
- Выбор типа лампы (ЛБ-40, ЛБ-80, ЛН-200, ДРЛ-400, МГЛ-250)
- Ввод нормированной освещённости E_н (50–1000 лк)
- Настройка коэффициента запаса K_з (1.0–2.0)
- Динамическая 3D-визуализация светильников в помещении

### 3️⃣ Лаборатория №3 — Шум и вибрация
**Цель:** рассчитать звукоизоляцию преграды по закону масс.

**Формулы:**
- Звукоизоляция: `R = 20 × lg(m × f) − 47.5` (дБ)
- Уровень звукового давления: `L = 20 × lg(P / P_0)` (дБ)

**Интерактивность:**
- Уровень источника шума (40–130 дБ)
- Частота звука (63–8000 Гц, октавная шкала)
- Выбор материала преграды (кирпич, бетон, стекло, дерево, сталь)
- Визуализация распространения звуковых волн и их ослабления

### 4️⃣ Лаборатория №5 — Защита от ЭМИ
**Цель:** рассчитать толщину экрана для защиты от электромагнитного излучения.

**Формулы:**
- ППЭ допустимая: `ППЭ_доп = N / T`, где `N = 2 Вт·ч/м²`
- Требуемое ослабление: `L = 10 × lg(ППЭ / ППЭ_доп)` (дБ)
- Толщина экрана: `δ = L / (8.69 × √(π × f × μ_а × γ))` (мм)

**Интерактивность:**
- Мощность источника P (10–1000 Вт)
- Частота излучения f (1–3000 МГц)
- Время облучения T (0.5–8 ч)
- Выбор материала экрана (сталь μ=200, медь μ=1, алюминий, пермаллой μ=70000)
- Визуализация ЭМ-волн до и после экрана

### 5️⃣ Лаборатория №6 — Поле УВЧ-диапазона
**Цель:** рассчитать напряжённость электрического поля от передатчика.

**Формулы:**
- Расстояние от антенны: `R = √(r² + H²)` (м)
- Напряжённость поля: `E = √(30 × P × G) / R` (В/м)
- Суммарная напряжённость (n передатчиков): `E_сум = √(E₁² + E₂² + ... + Eₙ²)`

**Интерактивность:**
- Мощность передатчика P (10–5000 Вт)
- Коэффициент направленности антенны G (1–100)
- Высота антенны H (5–100 м)
- Горизонтальное расстояние r (5–500 м)
- ПДУ напряжённости поля (1–50 В/м)
- Таблица напряжённости на различных расстояниях
- 3D-визуализация башни, антенны и распространяющихся волн

## 🎮 Использование

1. **Главное меню**: нажмите «Начать»
2. **Выбор темы**: выберите одну из 5 лабораторных работ
3. **Изучение теории**: пройдите все шаги (теория + 3D-сцены)
4. **Тестирование**: наберите ≥70% для допуска к лабораторной
5. **Лабораторная работа**: используйте интерактивные инструменты для расчётов

## 🛠️ Разработка

### Команды
```bash
npm run dev      # Dev-сервер с HMR
npm run build    # Production build (TypeScript → Vite)
npm run preview  # Запуск собранного build
npm run lint     # ESLint проверка
```

### Добавление новой лабораторной
1. Создайте файл `src/pages/labs/YourLab.tsx`
2. Добавьте тему в `src/data/themes.ts` (теория, сцены, квиз)
3. Создайте 3D-сцены в `src/components/scenes/YourScenes.tsx`
4. Зарегистрируйте сцены в `src/components/SceneViewer.tsx`
5. Добавьте роутинг в `src/pages/LabPage.tsx` (labMap)

### Обработка ошибок WebGL
Все `<Canvas>` обёрнуты в `<SafeCanvas>`, который:
- Ловит событие `webglcontextlost`
- Показывает overlay с кнопками перезапуска
- Позволяет восстановить рендеринг без перезагрузки страницы

## 📚 Справочная информация

### Нормы освещённости (СП 52.13330)
- Точная работа (I разряд): 500 лк
- Грубая работа (IV разряд): 200 лк
- Общее наблюдение (VIII разряд): 50 лк
- Офисные помещения: 300 лк
- Коридоры: 75 лк

### Допустимые уровни шума (СН 2.2.4/2.1.8.562-96)
- Производственные помещения: ≤80 дБА
- Порог слышимости: P₀ = 2×10⁻⁵ Па
- Болевой порог: >145 дБ

### ПДУ электромагнитных полей
- ППЭ для персонала (категория А): N/T, где N = 2 Вт·ч/м²
- Напряжённость поля УВЧ: зависит от частоты и времени

### Ионизирующее излучение (НРБ-99/2009)
- 100% летальная доза: 6 Гр
- Доза лучевой болезни: >1 Гр
- Фоновое облучение: ~300 мкЗв/год

## 🐛 Известные проблемы

### THREE.WebGLRenderer: Context Lost
Если видите это сообщение, попробуйте:
1. Обновить драйверы GPU (основная причина)
2. Закрыть другие GPU-интенсивные вкладки
3. Использовать кнопку «Перезапустить сцену» в overlay
4. Перезагрузить страницу браузера

### Производительность
- Для слабых GPU закройте другие приложения
- Множественные Canvas могут вызвать потерю контекста
- DPR ограничен до 1.5 для экономии памяти
